VidScaler

Eine benutzerfreundliche GUI-Anwendung zum Skalieren von Videos mit FFmpeg – inklusive Untertitel-Einbettung, automatischer Audio-Transkription und mehrsprachiger Übersetzung.

GitHub Repository Schnellstart

Features

Smart-Skalierung

Dropdown-Menü mit optimierten Skalierungswerten und automatischer Berechnung gerader Pixelwerte für maximale Kompatibilität.

Untertitel-Integration

Brennt .srt-Untertitel direkt ins Video ein – wahlweise Original, Übersetzung oder beides (dual mode).

Audio-Transkription

Erstellt automatisch SRT-Dateien aus Video-Audio mit OpenAI Whisper – mehrere Sprachen und Modelle verfügbar.

Mehrsprachige Übersetzung

OpenAI (beste Qualität), Google Translate (schnell) oder Whisper (lokal, nur Englisch) – 9 Sprachen unterstützt.

Dual Subtitles

Original-Untertitel oben, Übersetzung unten – perfekt für Sprachlerner und mehrsprachige Inhalte.

Smart Split

Videos automatisch in konfigurierbare Segmente aufteilen – mit einstellbarer Überlappung für nahtlose Übergänge.

Text-Exzerpt

Konvertiert SRT-Untertitel zu gut lesbaren Text- oder Markdown-Dokumenten – optional mit KI-Veredelung.

Workflow

1

Video auswählen & analysieren

Wähle dein Video aus und lass VidScaler die aktuelle Auflösung ermitteln. Optimierte Skalierungsoptionen werden automatisch berechnet.

2

Audio transkribieren (optional)

Klicke auf „Audio transkribieren“ um automatisch SRT-Untertitel zu erstellen. Whisper erkennt die gesprochene Sprache automatisch.

3

Übersetzung einrichten (optional)

Wähle Quell- und Zielsprache sowie die Übersetzungsmethode. OpenAI liefert die beste Qualität, Google ist schneller.

4

Aktion wählen & verarbeiten

Vier klare Buttons: „Video skalieren“, „Mit Original-Untertiteln“, „Mit Übersetzung“ oder „Mit Original + Übersetzung“ – fertig!

Installation

# Repository klonen
git clone https://github.com/ddumdi11/VidScalerSubtitleAdder.git
cd VidScalerSubtitleAdder

# Virtuelle Umgebung erstellen und aktivieren (Windows)
python -m venv .venv
.venv\Scripts\activate

# Dependencies installieren
pip install -r requirements.txt

# Anwendung starten
python vidscaler.py

# Oder einfach per Doppelklick:
start.bat

Voraussetzung: FFmpeg muss installiert und im PATH verfügbar sein.
FFmpeg herunterladen oder via Chocolatey: choco install ffmpeg
